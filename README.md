### Configuration

To customize your launcher, edit the `config.json` file located in the installation directory. Here you can set preferences like theme, default applications, and shortcuts.

### Example Configuration

```json
{
  "theme": "dark",
  "default_apps": [
    "browser",
    "editor",
    "terminal"
  ],
  "shortcuts": {
    "browser": "Ctrl+B",
    "editor": "Ctrl+E",
    "terminal": "Ctrl+T"
  }
}
```
